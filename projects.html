How to Add Your PowerPoint Presentation

To display your presentation on this webpage:

  1. Upload your PowerPoint to a service like Google Slides, Microsoft OneDrive, or SlideShare
  2. Use their embed feature to get the iframe code
  3. Replace the placeholder div with the embed code
  4. Update the download button link to point to your actual presentation file

Example embed code (replace with yours):

<iframe src="https://docs.google.com/presentation/d/your-presentation-id/embed" width="100%" height="100%" frameborder="0" allowfullscreen></iframe>
